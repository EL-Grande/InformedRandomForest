# InformedRandomForest
This is project describing an Informed Random Forest to Model Associations of Epidemiological Priors, Government Policies and Public Mobility. To run the experiments, please download the global datasets, and set the path to the data,  the code should be able to extract the data you want. In my case, I selected the Botswana data.

The link to the data is available here: https://ourworldindata.org/covid-mobility-trends?fbclid=IwAR2ziB. In my case, I just code  the script to extract my disered variables.
